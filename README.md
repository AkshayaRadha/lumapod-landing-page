Project Title : LumaPod – The Light That Understands You
LumaPod is a modern, responsive landing page designed for a smart ambient lighting product that adapts to your mood and environment.
This project highlights creativity, interactive design and clean UI development using HTML, CSS, JavaScript, and Bootstrap.

Overview : 
LumaPod is a creative landing page concept built to demonstrate how smart home lighting products can be presented through an elegant, interactive web interface. 
The website emphasizes smooth animations, responsive layouts, and modern design techniques to engage users visually and emotionally.
It showcases front-end design skills and focuses on blending functionality with aesthetics — making it ideal for product promotion or portfolio display.

Features : 
>> Responsive Design – Adapts perfectly across devices and screen sizes.
>> Dynamic Color Effects – Users can change lamp color in real-time.
>> Interactive Typing Animation – Highlights the tagline dynamically.
>> Glassmorphic Navbar – A sleek transparent effect for a modern look.
>> Animated Background – Gradient transitions and glowing particles for visual appeal.
>> Contact Section – Direct mail link for collaboration or feedback.


Tech Stack : 
Frontend: HTML5, CSS3, JavaScript
Framework: Bootstrap 5
Icons: Font Awesome
Typography: Google Fonts
Animation: CSS transitions & keyframes

Live demo : https://lumapod-landing-page.vercel.app/

Usage
>> Simply open the landing page and explore the interactive sections. 
>> The project demonstrates a product-based landing experience — you can customize text, images, and color schemes based on your product or brand concept.
